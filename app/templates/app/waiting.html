INSTRUCTIONS 

Now that you are a part of our community, you must know of some rules we follow around here.

General Rules and Instructions
  • The username that you would create would serve as your team name. Please share the credentials with your teammate (if any) and ask them to login using the same credentials. Immediately contact the team if you are facing any issues with registration. Our contact information is available here.
  • From the start of the event, 24 hours will be alloted to complete the challenges and boxes. The event will get over on 15 June, 2020 at 12:00 PM.
  • Each question has some points to it depending upon the difficulty. Taking a hint will deduce 10% of the points associated with the question.
  • There is no particular order of solving the questions.
  • Judging of the round will be based on two parameters. If there is a tie-breaker at points, the last submission time of the last solved question will be used as a secondary parameter to rank the users. The user having an earlier last-submission-timestamp will be ranked higher, in case of a tie-break.
  • Ranks can be viewed on the leaderboard page. The leaderboard is updated every time a challenge or box is solved or a hint is taken.
  • Flags found are of the format lakshya_CTF{some_text}. Some of the files may contain just the inner text within the brackets. Make sure you submit it in the format specified. Flag text is not case sensitive.
  • This is a competitive environment with the aim of learning cyber security and ethical hacking. Please do not share the flags & solutions with others.
  • Feel free to explore and read about the concept behind the problem.
Instructions for Solving Boxes
  • Each box is provisioned an IP address. You need to perform reconissance to get information about what services are running on the box.
  • Each box has its dedicated page, which can be seen by clicking on the machine card on the challenges' page. This dedicated page can provide useful information, and different approaches that you may follow to tackle the box.
  • There are two flags associated with each box - The user flag and the root flag. On successfully submitting the user flag, 40% of the total box points will be awarded, and on successful submission of the root flag, the remaining 60% points will be awarded.
  • Every box has a user - lakshya created on it.
  • When you are able to achieve command injection on the box (using some vulnerability or an exploit you crafted), you should read the user flag stored in the system at /home/lakshya/flag.txt. This flag is the password for the user - lakshya for that particular box. In order to escalate the system and gain root access, you can SSH into the machine with -
    "ssh lakshya@machine-ip" with the password as the user flag.
  • The root flag will be readable only by the root user. By correctly escalating privileges, you can find the root flag at /root/flag.txt.
  • You are requested not to delete any files on the system that can make the box inaccessible to other users. Every team should be given a fair and equal chance to takle the box.
  • After you have successfully rooted the box, you should delete your files and exploit programs, to ensure that others don't get hold of them.
  • To prevent unintended installations and bloatware on the system, we have disabled Internet access on the boxes. Any exploit files you wish to copy should be transferred over scp or any other medium that you are well-versed with.
  • Do not use Kernel/ distribution specific exploits to try and escalate privileges. All the kernels and distributions are updated to the latest versions. These kind of exploits are prone to brick the system and thus, making it inaccessible for other users.
Special Rules and Instructions
  • If you have any queries, or problems you can contact us here. Our team will be available for questions and problems throughout the event.
  • Performing denial of service attacks on the server will lead to disqualification.
  • Brute force attacks on the flag form is prohibited.
  • If you find a vulnerability in the website, feel free to use it for your advantage, until we patch it!
  • The write-ups for the challenges and boxes will be released after the CTF is over.
We expect you to adhere by the rules. Failure to do so might result in disqualification.
